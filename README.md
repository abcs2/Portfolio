# Página web de portfólio

## Navegação

A página é única e contém 3 botões que levam a diferentes seções, com o primeiro já estando automaticamente pressionado. Também há 2 links abaixo para sites externos.

## Seções

A seção 1 mostra informações gerais sobre mim, a seção 2 mostra projetos meus relacionados a programação, e a seção 3 mostra os sites que mais me ajudam na implementação dos meus programas.

Abaixo, há 2 links, um para o meu GitHub e outro para o meu Linkedin.

## Recursos CSS e JavaScript

No arquivo CSS, é utilizada uma grid para o posicionamento de todos os elementos principais da página, e também uma flexbox para o posicionamento dos botões na barra *aside*.

O recurso de Javascript permite mudar tanto a cor dos botões quanto o texto da barra *article*, o que permite ir para seções diferentes do portfólio sem necessitar de outros arquivos HTML.
